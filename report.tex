\begin{abstract}
We implement model checkers in Haskell for both the basic knowing-how logic $\mathcal{L}_{Kh}$ and the uncertainty-based knowing-how logic with regularity constraints $reg\text{-}\mathcal{L}^U_{Kh}$. 
For both logics, we validate the implementation by means of a range of tests using QuickCheck. 
In addition, to improve usability, we provide a parser for each logic.
For demo and testing purposes we implement a simple web-application.
\end{abstract}
